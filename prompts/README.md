# OCR Промпты

Эта папка содержит промпты для различных типов OCR распознавания.

## Файлы промптов

### `ocr_text.txt`
Используется для распознавания **TEXT** блоков.
- Простое распознавание текста
- Сохранение структуры

### `ocr_table.txt`
Используется для распознавания **TABLE** блоков.
- Распознавание таблиц
- Конвертация в Markdown формат

### `ocr_image_description.txt`
Используется для **IMAGE** блоков.
- Детальное описание изображения на русском языке
- Распознавание всех элементов
- Извлечение технической информации

### `ocr_full_page.txt`
Используется для распознавания **всей страницы** (режим "Вся страница").
- Комплексное распознавание
- Автоматическая структура документа

## Редактирование промптов

1. Откройте нужный `.txt` файл в любом текстовом редакторе
2. Измените текст промпта
3. Сохраните файл
4. Перезапустите приложение (или OCR операцию)

## Советы по созданию промптов

### Для TEXT блоков
- Укажите требования к структуре
- Определите формат вывода
- Уточните, нужны ли дополнительные элементы

### Для TABLE блоков
- Укажите формат таблицы (Markdown, HTML)
- Требования к заголовкам
- Обработка пустых ячеек

### Для IMAGE блоков
- Уровень детализации описания
- Язык описания
- Специфические элементы для поиска (схемы, графики, текст)
- Технические параметры

### Для полной страницы
- Обработка разных типов элементов
- Приоритеты распознавания
- Формат итогового документа

## Примеры промптов

### Минимальный (быстро, но менее точно)
```
Распознай текст.
```

### Детальный (медленнее, но точнее)
```
Выполни OCR распознавание текста на изображении.
Сохрани все форматирование: заголовки, списки, абзацы.
Распознай таблицы и преобразуй их в Markdown.
Если есть специальные символы или формулы - укажи их.
Проверь орфографию и исправь явные ошибки OCR.
Выведи результат в структурированном виде.
```

### Для технической документации
```
Распознай техническую документацию.
Обрати особое внимание на:
- Числовые параметры и единицы измерения
- Технические термины и обозначения
- Ссылки на стандарты и нормативы
- Таблицы с характеристиками
Сохрани точность всех числовых значений.
```

## Переменные в промптах

В будущих версиях планируется поддержка переменных:

```
Распознай {block_type} блок.
Язык документа: {language}
Формат вывода: {output_format}
```

## Отладка

Если OCR работает неправильно:
1. Проверьте содержимое промпта
2. Уберите специальные символы
3. Упростите инструкции
4. Тестируйте на простых примерах
