# Распознавание изображений в документах

## Обработка изображений с Chandra OCR

Chandra OCR автоматически обрабатывает изображения в документах:
- Извлекает изображения из документа
- Сохраняет их в папку `images/`
- Включает описания в Markdown вывод

## Использование

1. Откройте PDF: **Файл → Открыть PDF**
2. Запустите OCR: **Инструменты → Запустить OCR** (Ctrl+R)
3. Выберите:
   - **Вся страница (автоматическая структура)**
   - Метод: **HuggingFace** или **vLLM сервер**
4. Результат:
   - Markdown файл с текстом, таблицами, изображениями
   - Папка `images/` с извлеченными изображениями
   - Полное сохранение структуры документа

## Структура результата

```markdown
# Страница 1

<table>...</table>

![Схема вентиляции](images/page0_image_001.png)

Текст документа продолжается...
```

## Технические детали

### Файл: `app/ocr.py`

**Функция**: `run_chandra_ocr_full_document()`
- Обрабатывает каждую страницу через Chandra CLI
- Автоматически извлекает изображения (флаг `--include-images`)
- Копирует изображения в выходную директорию
- Объединяет результаты в единый Markdown документ

## Возможности Chandra OCR

- ✅ Таблицы в Markdown формате
- ✅ Формы и структура
- ✅ Рукописный текст
- ✅ Изображения и диаграммы
- ✅ Полное сохранение layout документа

## Ограничения

- Требует установленного Chandra OCR
- GPU рекомендуется для ускорения
- Время обработки зависит от сложности документа
