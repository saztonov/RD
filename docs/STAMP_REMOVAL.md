# Удаление электронных штампов

## Функционал

Система позволяет:
- Анализировать структуру PDF (аннотации, изображения, контейнеры)
- Удалять ненужные элементы (штампы, водяные знаки)
- Предпросматривать результат
- Сохранять очищенный PDF

## Использование

### Способ 1: Через меню
1. **Открыть PDF**: Файл → Открыть PDF
2. **Запуск**: Инструменты → Удалить штампы (Ctrl+D)
3. **Выбор**: Отметить элементы для удаления
4. **Удаление**: Нажать "🗑️ Удалить выбранные"
5. **Предпросмотр**: Проверить результат
6. **Применение**: "✓ Применить и загрузить"

### Способ 2: Через панель
1. **Открыть PDF**: Файл → Открыть PDF
2. **Кнопка**: Нажать "🗑️ Удалить штампы" в правой панели
3. Далее как в способе 1

## Типы элементов

- **Аннотации**: Штампы, комментарии, подписи (обычно это штампы)
- **Изображения**: Встроенные изображения (логотипы, печати)
- **Контейнеры**: Form XObjects (сложные штампы)

## Фильтры

Чекбоксы для отображения:
- ☑️ Аннотации
- ☑️ Изображения
- ☑️ Контейнеры

## Workflow

```
1. Открыть PDF (быстро)
   ↓
2. Инструменты → Удалить штампы
   ↓
3. Анализ структуры (с прогрессом)
   ↓
4. Выбор элементов
   ↓
5. Удаление → Предпросмотр
   ↓
6. Сохранение → Загрузка в редактор
```

## Производительность

- **Малые PDF (<10 стр)**: анализ мгновенно
- **Средние PDF (10-50 стр)**: прогресс-бар
- **Большие PDF (>50 стр)**: анализ может занять ~1 мин

## Горячие клавиши

- `Ctrl+D` - Открыть диалог удаления штампов
- `Delete` - Удалить выбранный элемент в дереве
