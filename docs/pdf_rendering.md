### 2. `render_page_to_image(doc: fitz.Document, page_index: int, zoom: float = 2.0) -> PIL.Image.Image`

–†–µ–Ω–¥–µ—Ä–∏—Ç –æ–¥–Ω—É —Å—Ç—Ä–∞–Ω–∏—Ü—É PDF –≤ –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–µ PIL.

**–ü–∞—Ä–∞–º–µ—Ç—Ä—ã:**

- `page_index` (int) ‚Äî –∏–Ω–¥–µ–∫—Å —Å—Ç—Ä–∞–Ω–∏—Ü—ã (–Ω–∞—á–∏–Ω–∞—è —Å 0)

  - `1.0` = 72 DPI (–Ω–∞—Ç–∏–≤–Ω–æ–µ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ PDF)
  - `2.0` = 144 DPI (—Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –¥–ª—è —á—Ç–µ–Ω–∏—è)
  - `3.0` = 216 DPI (–≤—ã—Å–æ–∫–æ–µ –∫–∞—á–µ—Å—Ç–≤–æ)

**–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç:**

**–ò—Å–∫–ª—é—á–µ–Ω–∏—è:**
- `IndexError` ‚Äî –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã–π –∏–Ω–¥–µ–∫—Å —Å—Ç—Ä–∞–Ω–∏—Ü—ã
- `ValueError` ‚Äî –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã–π zoom (‚â§ 0)
- `Exception` ‚Äî –æ—à–∏–±–∫–∏ —Ä–µ–Ω–¥–µ—Ä–∏–Ω–≥–∞

**–í–∞–∂–Ω–æ:** Zoom –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –æ–¥–∏–Ω–∞–∫–æ–≤–æ –ø–æ X –∏ Y –¥–ª—è —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –ø—Ä–æ–ø–æ—Ä—Ü–∏–π.

### –ü—Ä–∏–º–µ—Ä 3: –ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—è —Å GUI (PySide6)

```python
from PySide6.QtGui import QPixmap, QImage
from app.pdf_utils import open_pdf, render_page_to_image

def load_pdf_page_to_qt(pdf_path: str, page_idx: int) -> QPixmap:
    """–ó–∞–≥—Ä—É–∑–∏—Ç—å —Å—Ç—Ä–∞–Ω–∏—Ü—É PDF –≤ QPixmap –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –≤ Qt"""
    doc = open_pdf(pdf_path)
    pil_image = render_page_to_image(doc, page_idx, zoom=2.0)
    doc.close()
    
    # –ö–æ–Ω–≤–µ—Ä—Ç–∞—Ü–∏—è PIL ‚Üí QPixmap
    img_bytes = pil_image.tobytes("raw", "RGB")
    qimage = QImage(
        img_bytes,
        pil_image.width,
        pil_image.height,
        pil_image.width * 3,
        QImage.Format_RGB888
    )
    return QPixmap.fromImage(qimage)
```
