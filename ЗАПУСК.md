# Команды запуска

## Сервер (dev режим)

```bash
docker compose -f docker-compose.remote-ocr.dev.yml up --build
```

Или без Docker:
```bash
cd services/remote_ocr
uvicorn services.remote_ocr.server.main:app --host 0.0.0.0 --port 8000 --reload
```

## Клиент

```bash
python -m app.main
```

## Настройки .env

```
# Remote OCR сервер
REMOTE_OCR_BASE_URL=http://localhost:8000
REMOTE_OCR_API_KEY=

# OCR движки
OPENROUTER_API_KEY=your_key_here
DATALAB_API_KEY=your_key_here

# R2 Storage (для загрузки результатов)
R2_ACCOUNT_ID=your_account_id
R2_ACCESS_KEY_ID=your_access_key
R2_SECRET_ACCESS_KEY=your_secret_key
R2_BUCKET_NAME=rd1
R2_PUBLIC_URL=  # Публичный URL R2 bucket (например: https://pub-xxxxx.r2.dev)
```

## Проверка работы

```bash
curl http://localhost:8000/health
```
